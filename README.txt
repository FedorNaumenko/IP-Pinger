# PingerApp

PingerApp is a Python application that allows you to ping multiple IP addresses from a list.

## Features

- Load IP addresses from a text file
- Ping multiple IP addresses concurrently
- Display the ping results in a table
- Remember the last opened file path
- Sort the table by IP address or name

## Usage

1. Run `main.py` to start the application.
2. Click on the 'Load IPs' button to load IP addresses from a text file.
3. Click on the 'Start Pinging' button to start pinging the loaded IP addresses.
4. The ping results will be displayed in the table.
